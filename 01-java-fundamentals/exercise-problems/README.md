# Java Fundamentals — Exercise Problems

This folder contains practice problems for the **Java Fundamentals** section.

The goal of these exercises is to strengthen understanding of basic Java concepts before moving to advanced programming topics and Data Structures & Algorithms.

## Topics Covered (So Far)

- Variables and Data Types
- Operators
- Input / Output

Later this folder will also include exercises for:

- Conditionals
- Loops

Each topic contains **10 coding problems** divided by difficulty:

| Difficulty | Count |
| ---------- | ----- |
| Easy       | 3     |
| Moderate   | 4     |
| Difficult  | 3     |

Total per topic: **10 problems**

---

# 1. Variables & Data Types (10 Problems)

### 🟢 Easy

**1.** Write a program that declares variables of types `int`, `double`, `char`, and `boolean`, assigns values to them, and prints them.

**2.** Write a program to store your **name, age, and height** in variables and print them.

**3.** Declare two integer variables and print their **sum and difference**.

---

### 🟡 Moderate

**4.** Write a program that declares three variables: `length`, `width`, and `height`, and calculates the **volume of a cuboid**.

**5.** Write a program that demonstrates **type casting** by converting a `double` value to an `int`.

**6.** Write a program that stores the **ASCII value of a character** in an integer and prints both.

**7.** Declare multiple variables in a single line and print their values.

---

### 🔴 Difficult

**8.** Write a program that swaps the values of two variables using a **third variable**.

**9.** Write a program that swaps two numbers **without using a third variable**.

**10.** Write a program that converts **temperature from Celsius to Fahrenheit**.

Formula:
F = (C × 9/5) + 32

---

# 2. Operators (10 Problems)

### 🟢 Easy

**1.** Write a program that takes two numbers and prints the results of:
/
%

**2.** Write a program that checks whether a number is **even or odd** using the modulus operator.

**3.** Write a program that compares two numbers and prints which one is larger.

---

### 🟡 Moderate

**4.** Write a program that calculates the **average of three numbers**.

**5.** Write a program that checks if a number lies between **10 and 50** using logical operators.

**6.** Write a program that demonstrates the difference between:
x++
++x

**7.** Write a program that evaluates a logical expression such as:
(a > b && b > c)
and prints the result.

---

### 🔴 Difficult

**8.** Write a program that checks whether a number is divisible by **both 3 and 5**.

**9.** Write a program that calculates the **area of a circle** using the formula:
Area = π × r × r

Use `Math.PI`.

**10.** Write a program that uses **compound assignment operators** (`+=`, `-=`, `*=`, `/=`) on a variable and prints the results.

---

# 3. Input / Output (10 Problems)

### 🟢 Easy

**1.** Write a program that takes the user's **name as input** and prints:
Hello <name>

**2.** Write a program that takes an integer input and prints:
You entered: X

**3.** Write a program that takes two numbers as input and prints their **sum**.

---

### 🟡 Moderate

**4.** Write a program that takes **three numbers from the user** and prints their average.

**5.** Write a program that asks for **name and age** and prints:
<Name> is <age> years old.

**6.** Write a program that takes **radius of a circle** from the user and calculates the area.

**7.** Write a program that takes **length and width** as input and prints the **area of a rectangle**.

---

### 🔴 Difficult

**8.** Write a program that takes **a number from the user** and prints its **square and cube**.

**9.** Write a program that asks for **two numbers and an operator (+, -, \*, /)** and prints the result.

**10.** Write a program that asks the user for **five numbers and prints their sum and average**.

# Recommended Practice Method

1. Attempt problems **without looking at solutions**.
2. Write the program completely and compile it.
3. Test the program with different inputs.
4. Refactor the code for clarity.
5. Commit completed solutions to your repository.
