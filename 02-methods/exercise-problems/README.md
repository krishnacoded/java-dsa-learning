# Java Methods – Practice Problems

Solve the following problems using **separate methods** wherever possible.  
Avoid writing all logic inside `main()`.

---

## 1. Even or Odd using Method

Write a method `isEven(int n)` that returns `true` if the number is even and `false` otherwise.

Example:
Input: `7`  
Output: `Odd`

---

## 2. Maximum of Three Numbers

Create a method:
maxOfThree(int a, int b, int c)
that returns the largest number among the three.

Example:
Input: `3 9 5`  
Output: `9`

---

## 3. Factorial using Method

Write a method:
factorial(int n)

that returns the factorial of a number.

Example:
Input: `5`  
Output: `120`

---

## 4. Prime Number Check

Create a method:
isPrime(int n)

that returns `true` if the number is prime, otherwise `false`.

Example:
Input: `11`  
Output: `Prime`

---

## 5. Reverse a Number

Write a method:
reverseNumber(int n)

that returns the reversed number.

Example:
Input: `1234`  
Output: `4321`

---

## 6. Palindrome Number

Write a method:
isPalindrome(int n)

that checks if a number is a palindrome.

Example:
Input: `121`  
Output: `Palindrome`

---

## 7. Sum of Digits

Create a method:
sumOfDigits(int n)
that returns the sum of digits of a number.

Example:
Input: `245`  
Output: `11`

---

## 8. Power of a Number

Write a method:
power(int base, int exponent)
that returns `base^exponent`.

Example:
Input: `2 5`  
Output: `32`

---

## 9. Print Multiplication Table

Create a method:
printTable(int n)
that prints the multiplication table of `n` up to 10.

Example:
Input: `5`

Output:5 x 1 = 5
5 x 2 = 10
...
5 x 10 = 50

---

## 10. Count Digits in a Number

Write a method:
countDigits(int n)
that returns the number of digits in the number.

Example:
Input: `45678`  
Output: `5`

---

# Rules

- Use **methods for the main logic**
- `main()` should mostly handle **input/output**
- Avoid duplicating logic
- Try both **void methods and return methods**
